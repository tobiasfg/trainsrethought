# Auxiliary Topics Overview

The auxiliary docs contain long-form articles on important or advanced topics related to the modding API. They can be relevant to either or both prototype/runtime stages. The most important of these for any modder to understand is the Data Lifecycle.

The docs also include a list of external topics from the wiki. These cover information that is partially related to modding, but also relevant for players of the game.

Additionally, this section of the docs contains the defines, which are a set of enums used by various parts of the API, in both prototype and runtime stages.

## Key Concepts
- Data Lifecycle ties settings, prototype, and runtime stages together.
- Defines are shared enums used across the API.
- External wiki topics may affect both stage behavior and gameplay.

## Where to Look Next

- `prototype/overview.md` for base data structures.
