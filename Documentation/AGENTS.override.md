# Documentation Guide

These docs are organized by modding stage: settings, prototype, runtime, and auxiliary topics. Start with `README.md`, then read the stage overviews in order. Use `settings/mod-settings.md` for the full settings tutorial and `prototype/data-raw/index.md` to browse base game prototype categories. The `_sources` folder contains original copied files and should be treated as read-only archival sources. If you hit missing or incomplete sections, ask the user to provide or locate the additional source documentation.
