# TrainsRethought Project

This project is a Factorio mod that reworks how trains transport items and fluids to make rail logistics competitive again in the Space Age era. The core change is turning cargo wagons into flatbeds that hold placeable chests instead of acting as fixed containers. A new inserter type will lift, place, or swap chests on wagons so belts or inserters can interact with chest inventories on the train. The mod targets restoring train viability versus upgraded belts and machines by changing loading mechanics and capacity flexibility.

Documentation lives in `Documentation/README.md` with stage-based references and prototype catalog slices. Some Factorio wiki/API content may be missing because only a subset was copied; if critical details are absent, ask the user to locate or add the missing source.
